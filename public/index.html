单选L2 · 2024-03 · 第 5 题中等
下面这段 C++ 代码,运行后输出是多少?
int sum = 0;
for (int i = 1; i <= 3; i++)
    sum += i;
cout << sum;
A3
B5
C6✓
D10
答对了
循环依次把 i = 1、2、3 加进 sum,所以 sum = 6。注意 for 条件 i <= 3 会执行到 i 等于 3。
单选L3 · 2025-06 · 第 7 题中等
下面 C++ 代码,运行后输出是多少?
int a = 5;
cout << (~a);
A-5
B-6✓
C5
D6
答对了
按位取反满足 ~a = -a-1,所以 ~5 = -6。补码下这是三级常考的小陷阱。
单选L4 · 2024-09 · 第 3 题中等
下面 C++ 代码,运行后输出是多少?
int a = 10;
int *p = &a;
*p = 20;
cout << a;
A10
B20✓
C报错
D不确定
答对了
p 指向 a,*p = 20 改的就是 a 本身,所以输出 20。指针操作的是地址里存的值。
单选L1 · 2023-12 · 第 8 题简单
下面 C++ 代码,运行后输出是?
char c = 'A';
cout << char(c + 1);
AA
BB✓
C66
D1
答对了
字符按 ASCII 参与运算,'A'+1 得 66,再转回字符就是 'B'。
